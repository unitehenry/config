# Setting Up My Environment

> Start from a fresh zsh installation with Homebrew

## Load Configuration

```sh
$ git clone https://github.com/unitehenry/config

$ mv config/.zshrc ~

$ mv config/.vimrc ~

$ mv config/.tmux.conf ~
```

## Install Tools

```sh
$ brew install fzf

$ brew install tmux

$ brew install vim
```

## Browser

```sh
$ brew cask install chromium
```

## Vim Plugins

```
~/.vim/pack/plugins/start/{package-name}
```
